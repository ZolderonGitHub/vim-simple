## Features:

- AutoCompletion(lsp and buffer completion)
- Fuzzy file finding
- File browser
- Compilation(default build file: build.bat)
- Airline
- Some good colorschemes

Leader key is space.

- Space + e = open file browser
- Space + b = compile;
- Space + (h, j, k, l) = move to a different buffer when the window is splited
